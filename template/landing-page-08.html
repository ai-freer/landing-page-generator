部署 NebulaAI 需要多长时间？

标准部署通常在 2-4 周内完成，包括数据源对接、模型调优和团队培训。我们提供专属的客户成功团队全程陪伴。

支持哪些数据库和数据源？

支持 MySQL、PostgreSQL、MongoDB、Kafka、S3、Snowflake 等 200+ 数据源，也支持自定义 REST API 接入。

数据安全如何保障？

通过 SOC 2 Type II 认证，支持端到端 AES-256 加密、VPC 私有部署、RBAC 权限管理，满足金融级合规要求。

是否需要机器学习背景？

完全不需要。NebulaAI 内置 AutoML 引擎，业务团队可以通过可视化界面完成建模、训练和部署的全流程。

如何计费？

提供按数据量和用户数的弹性计费方案。初创企业可使用免费版（支持 100 万数据点/月），企业版支持无限数据量。

能否与现有 BI 工具集成？

完全兼容 Tableau、Power BI、Looker 等主流 BI 工具，也提供开放 API 和 SDK 供深度集成。
