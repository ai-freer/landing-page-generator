# template-01: 经典Hero+Features

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**

**设计理念**：
以Hero区域为核心，通过大标题、副标题和CTA按钮快速吸引用户注意力，随后通过功能网格展示产品核心价值，最后通过用户评价建立信任。这是最经典的SaaS产品落地页结构。

**结构组成**：
1. Hero区域（全屏渐变背景，居中对齐）
   - 产品名称（H1，64px）
   - 产品标语（20px）
   - CTA按钮（白色背景，紫色文字，50px圆角）
   - Hero图片（可选，20px圆角，带阴影）
   - 作用：快速吸引用户注意力

2. Features区域（3列自适应网格布局）
   - 区域标题（48px，居中）
   - 副标题（18px，居中）
   - 功能卡片（3-6个）
     - 功能图标（紫色渐变背景，64x64px）
     - 功能标题（24px，紫色）
     - 功能描述（16px）
   - 悬停效果：卡片上移、紫色边框、阴影增强
   - 作用：展示产品核心价值

3. Testimonials区域（3列自适应卡片布局）
   - 区域标题（48px，居中）
   - 副标题（18px，居中）
   - 用户评价卡片（3-6个）
     - 用户评价（17px，斜体，带引号装饰）
     - 作者信息（头像+姓名+身份）
   - 作用：建立信任

4. FAQ区域（可选，折叠式问答列表）
   - 区域标题（48px，居中）
   - FAQ列表（带分隔线）
   - 作用：解答常见问题

5. CTA区域（可选，渐变背景）
   - CTA标题（48px）
   - 副标题（18px）
   - CTA按钮（白色背景）
   - 作用：引导转化

6. Footer区域（深色背景）
   - 品牌名称（32px）
   - 品牌标语
   - 版权信息
   - 社交媒体链接
   - 作用：页面收尾

**核心特点**：
- ✨ 紫色渐变：#667eea → #764ba2，视觉吸引力强
- 📐 3列网格：自适应布局，响应式友好
- 🎯 信任建立：用户评价区增强可信度
- 💡 结构清晰：Hero→Features→Testimonials→CTA的完整流程
- 🌟 微交互：卡片悬停、按钮动画、阴影过渡

**配置要点**：

**必填字段**：
```yaml
product:
  name: 产品名称
  tagline: 一句话价值主张
  description: 产品描述
```

**功能网格**：
```yaml
features:
  - title: 功能1
    description: 简短描述（1-2句话）
  - title: 功能2
    description: 简短描述
  # ... 建议3-6个功能
```

**可选字段**：
```yaml
mock_data:
  testimonials:
    - quote: 评价内容（1-2句话）
      author: 作者信息（姓名+身份）
  # ... 建议3-6个评价

faq:
  - question: 常见问题
    answer: 详细答案
  # ... 建议3-6个问题

cta:
  title: 底部CTA标题
  subtitle: 副标题
  button_text: 按钮文字
```

**视觉风格建议**：
- 🎨 配色方案：
  - 主色调：紫色渐变（#667eea → #764ba2）
  - Hero背景：紫色渐变
  - 按钮背景：白色
  - 文字颜色：#1a1a1a（深黑）、#666（中灰）、#999（浅灰）
  - 浅色背景：#f9f9f9
- 📝 字体：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- ⚪ 圆角：12-16px中圆角（卡片）、50px大圆角（按钮）、20px（Hero图片）
- 💫 阴影：柔和阴影（0 4px 12px rgba(0,0,0,0.08)），悬停增强
- 🎯 对比：渐变背景+白色文字形成强对比
- 🌟 动画：卡片上移（-8px）、按钮缩放（1.02）、阴影过渡

**适用场景**：
- ✅ SaaS平台（功能展示）
- ✅ 工具类软件（快速推广）
- ✅ 在线服务（建立信任）
- ❌ 不适合需要深色主题的产品（推荐template-08）
- ❌ 不适合奢侈品产品（推荐template-15）

## 美学重点
- 🎯 **紫色渐变Hero**：大色块渐变背景创造视觉冲击力，快速吸引用户注意力
- 📐 **3列网格布局**：功能卡片规整排列，信息层次清晰，展示产品丰富性
- ✨ **微交互设计**：卡片悬停上移、阴影增强，创造"活"的页面感觉
- 💜 **紫色品牌色**：统一的紫色视觉语言，强化品牌识别度
- 🌟 **渐变CTA按钮**：白色背景+紫色渐变边框，引导转化
- 💡 **信任建立**：用户评价区增强可信度，完整Hero→Features→Testimonials流程
