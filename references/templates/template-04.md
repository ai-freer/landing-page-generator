# template-04: 双列布局+视频

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**


**设计理念**：
内容与视频结合，信息传达丰富，特别适合教育内容和课程展示。采用交替分屏布局和视频演示，让用户通过观看了解产品。

**结构组成**：
1. Hero区域（深蓝渐变背景，居中对齐）
   - 产品名称（H1，64px）
   - 副标题（22px）
   - CTA按钮组（金色背景+白色描边）
   - 作用：快速介绍产品

2. Stats区域（可选，数据展示）
   - 统计数据网格（2-4个）
     - 统计数字（56px，大字号）
     - 统计标签（16px，大写）
   - 作用：展示数据成果

3. Features区域（可选，网格布局）
   - 区域标题（48px，居中）
   - 副标题（18px，居中）
   - 功能卡片（2-4个）
     - 功能图标（深蓝渐变背景，56x56px）
     - 功能标题（24px）
     - 功能描述（16px）
   - 悬停效果：卡片上移、阴影增强
   - 左侧金色边框装饰
   - 作用：补充功能说明

4. Content Sections（可选，交替分屏布局）
   - 2-4个内容段落
   - 每个段落：
     - 左侧：内容标题+描述
     - 右侧：媒体占位符（图片或视频）
   - 布局：交替左右（左文右图→右文左图）
   - 作用：丰富信息展示

5. Video Section（可选，视频区域）
   - 视频标题
   - 视频容器（YouTube/Vimeo嵌入）
   - 作用：展示视频内容

6. CTA区域（深蓝渐变背景）
   - CTA标题
   - 副标题
   - CTA按钮
   - 作用：引导转化

7. Footer区域（深色背景）
   - 品牌名称
   - 版权信息

**核心特点**：
- 📹 视频优先：视频演示，直观展示
- 📐 交替布局：左右分屏，视觉节奏感
- 🎓 内容丰富：适合教育内容和课程
- 💡 信息传达：图文视频结合，信息密度高
- 🌟 海军蓝主题：#1e3a5f主色，金色点缀（#d4af37）

**配置要点**：

**必填字段**：
```yaml
product:
  name: 产品名称
  tagline: 一句话价值主张
```

**可选字段**：
```yaml
hero:
  stats:
    - value: 统计数值
      label: 统计标签
  # ... 建议2-4个统计数据

content_sections:
  - title: 章节标题
    description: 详细描述
    media_placeholder: 媒体占位符
  # ... 建议2-4个段落

video_embed: 视频嵌入URL

features:
  - title: 功能1
    description: 简短描述
  # ... 建议2-4个功能

cta:
  button_text: 按钮文字
```

**视觉风格建议**：
- 🎨 配色方案：
  - 主色调：深海军蓝（#1e3a5f）、次蓝色（#2c5282）
  - 强调色：金色（#d4af37）
  - Hero背景：深蓝渐变
  - 文字颜色：#1a1a1a（深黑）、#4a5568（中灰）、#a0aec0（浅灰）
  - 浅色背景：#f8fafc
- 📝 字体：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- ⚪ 圆角：4px小圆角（按钮）、8px（卡片）、12px（Hero图片）
- 💫 阴影：柔和阴影（0 2px 12px rgba(0,0,0,0.08)），悬停增强
- 🎯 对比：深色背景+白色文字，金色点缀
- 🌟 装饰：功能卡片左侧金色边框（4px）
- 🎨 渐变：Hero背景深蓝渐变，右上角金色光晕

**适用场景**：
- ✅ 在线课程（教育内容）
- ✅ 内容创作工具（视频演示）
- ✅ 教育平台（课程展示）
- ❌ 不适合功能密集的产品（推荐template-01）
- ❌ 不适合快速转化的产品（推荐template-02）

## 美学重点
- 🎬 **视频演示核心**：视频作为主要内容，动态展示产品价值
- 📐 **交替分屏布局**：左图右文、右图左文交替，节奏感强，避免视觉疲劳
- 🎨 **深蓝+金色配色**：深海军蓝背景+金色点缀，营造专业高级感
- 🌟 **视频Hero区域**：首屏视频展示，快速传达产品核心功能
- 💫 **金色装饰细节**：功能卡片左侧金色边框，精致而优雅
- 🎯 **教育内容导向**：清晰的课程结构，适合学习型产品展示
