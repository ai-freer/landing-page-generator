# template-08: 深色沉浸式

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**


**设计理念**：
引入强对比和深色沉浸式设计手法，视觉冲击力显著提升。通过"亮-暗-亮"的节奏切换，制造视觉上的"呼吸"和"聚焦"。关键信息区域使用深色背景+反白文字，这是典型的强调型设计，用来锁定用户的注意力。整体能量场像"IMAX电影屏幕"，营销感最强。

## 结构组成

### 1. Hero区域（居中布局，白色背景）
- **结构定义**：居中对齐的开篇区域
- **视觉特征**：
  - 居中对齐的大标题（72px）
  - 紫色渐变文字效果
  - 居中大图（可选）
  - 紫色渐变CTA按钮
- **作用**：保持开篇的清爽和专业感

### 2. Immersive Feature区域（全宽深色背景）
- **结构定义**：巨大的视觉断点
- **视觉特征**：
  - 巨大的深蓝到紫色的渐变（#1a1a2e → #16213e → #0f3460）
  - 左右分屏布局：左侧反白文字（玻璃态效果），右侧插图
  - 玻璃态文字容器：半透明白色背景+backdrop-filter模糊
  - 插画边缘自然融入深色背景（无生硬边框）
- **作用**：锁定用户注意力，强调核心价值

### 3. Features Grid区域（白色背景）
- **结构定义**：3×2网格布局
- **视觉特征**：
  - 简约卡片设计：边框+悬停效果
- **作用**：从深海浮出水面，缓解视觉疲劳

### 4. Testimonials区域（白色背景）
- **结构定义**：3列卡片布局
- **视觉特征**：
  - 与功能网格连在一起，形成较长的"理性信息阅读区"（白底区域）
- **作用**：让用户冷静下来看具体的评价

### 5. FAQ区域（极光渐变背景）
- **结构定义**：3列网格布局（平铺展示，非折叠列表）
- **视觉特征**：
  - 全息/极光渐变：淡粉、淡紫、淡蓝交织（#667eea → #764ba2 → #f093fb → #f5576c → #4facfe）
  - 动态渐变动画（15秒循环）
  - 白色半透明卡片悬浮在极光背景上
- **氛围**：现代感、高级、轻盈，与深色区域形成鲜明对比

### 6. Bottom CTA区域（深蓝渐变背景）
- **结构定义**：居中布局
- **视觉特征**：
  - 呼应Immersive Feature的深色背景
  - 居中大标题+居中白色按钮
- **作用**：利用视觉重音引导转化

### 7. Footer区域（白色背景）
- **结构定义**：极简白底黑字链接
- **作用**：平衡整体视觉，避免过重

## 核心特点
- 🎬 IMAX级视觉冲击：深色高对比，能量场强
- 🎨 视觉重音：关键区域（Immersive Feature + CTA）使用深色背景
- 🌌 沉浸式体验：深蓝渐变 + 玻璃态 + 极光渐变
- 💡 节奏感："亮-暗-亮"的呼吸式视觉流动
- 🔥 营销感最强：强迫用户视线聚焦核心价值
- ✨ 高级感：玻璃态、动态渐变、反白文字

## 配置要点

### 必需字段
- product.name
- product.tagline

### Hero配置（可选）
```yaml
hero:
  image_url: 首屏大图URL（可选）
```

### Immersive Section配置（新增）
```yaml
immersive_section:
  title: 沉浸式功能标题
  description: 详细描述...
  image_url: 插图URL（可选）
```

### 功能网格
```yaml
features:
  - title: 功能1
    description: 简短描述
  - title: 功能2
    description: 简短描述
  # ... 共6个功能
```

### 可选字段
```yaml
testimonials:
  - name: 用户姓名
    role: 身份/职业
    content: 评价内容
    avatar_url: 头像URL
  # ... 共3个评价

faq:
  - question: 常见问题1
    answer: 详细答案

cta:
  title: 底部CTA标题
  subtitle: 补充说明
  button_text: 按钮文字

footer:
  tagline: 品牌标语
  social_links:
    - name: Twitter
      url: https://twitter.com/...
```

## 视觉风格建议
- 🎨 配色方案：
  - 深蓝渐变：#1a1a2e → #16213e → #0f3460（Immersive、CTA）
  - 极光渐变：#667eea → #764ba2 → #f093fb → #f5576c → #4facfe（FAQ，动态）
  - 白色：#ffffff（Hero、Features、Testimonials、Footer）
- 💜 主色调：紫色渐变（#667eea → #764ba2）
- 📝 字体：标题超粗（700-800），正文清晰
- 🖼️ 图片：推荐使用透明PNG/SVG，边缘处理好避免白边
- 🌟 玻璃态：半透明白色背景+backdrop-filter模糊
- ⚪ 圆角：24px大圆角
- 💫 阴影：深色区域使用强阴影（0 40px 80px）
- 🎯 对比：深色背景上的白色文字，极光背景上的白色卡片
- 🎬 动画：极光渐变15秒循环动画

## 使用建议
- ✅ 适合需要强烈视觉冲击的产品
- ✅ 适合企业级软件、高科技产品
- ✅ 适合需要建立品牌信任的产品
- ✅ 适合高价值产品（B2B）
- ❌ 不适合极简风格产品（推荐template-05）
- ❌ 不适合清新自然风格（推荐template-06）

- 🎬 **IMAX级视觉冲击**：深色高对比，能量场强，锁定用户注意力
- 🌌 **沉浸式体验**：深蓝渐变+玻璃态+极光渐变，营造沉浸氛围
- 💡 **节奏感**："亮-暗-亮"的呼吸式视觉流动，引导用户情绪
- 🔥 **营销感最强**：强迫用户视线聚焦核心价值，强化品牌印象

- 🌟 **极光渐变**：全息/极光渐变，动态动画，增强视觉层次
- 🎯 **视觉重音**：关键区域使用深色背景，制造视觉断点
