# template-02: 产品展示+CTA

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**


**设计理念**：
产品视觉优先，突出产品外观和价格，配合分屏布局和强CTA引导转化。特别适合实物商品和订阅服务。

**结构组成**：
1. Hero区域（浅色渐变背景，左右分屏布局）
   - 左侧：产品信息
     - 产品名称（H1，56px）
     - 产品标语（20px）
     - 产品描述（16px）
     - CTA按钮（蓝色背景，12px圆角）
   - 右侧：产品图片（大图展示，20px圆角）
   - 作用：突出产品外观

2. Features区域（可选，网格布局）
   - 区域标题（48px，居中）
   - 副标题（18px，居中）
   - 功能卡片（2-4个）
     - 功能图标（蓝色渐变背景，56x56px）
     - 功能标题（22px）
     - 功能描述（16px）
   - 悬停效果：卡片上移、蓝色边框、阴影增强
   - 作用：补充产品功能

3. Testimonials区域（可选，卡片布局）
   - 区域标题（48px，居中）
   - 副标题（18px，居中）
   - 用户评价卡片
   - 作用：建立信任

4. CTA区域（全屏深色背景）
   - CTA标题（48px）
   - 副标题（18px）
   - 价格展示（突出显示）
   - 购买按钮（大号CTA）
   - 作用：引导转化

5. Footer区域（深色背景）
   - 品牌名称
   - 版权信息
   - 社交媒体链接

**核心特点**：
- 🖼️ 产品优先：大图展示，视觉冲击强
- 💰 价格突出：清晰的价格展示
- 🎯 转化导向：强CTA引导购买
- 📐 分屏布局：左文右图，平衡感强
- 🌟 蓝色主题：#4299e1主色，专业感强

**配置要点**：

**必填字段**：
```yaml
product:
  name: 产品名称
  tagline: 一句话价值主张
  image: 产品图片URL
  price: 产品价格
```

**可选字段**：
```yaml
features:
  - title: 功能1
    description: 简短描述
  # ... 建议2-4个功能

mock_data:
  testimonials:
    - quote: 评价内容
      author: 作者信息
  # ... 建议3-6个评价

cta:
  button_text: 按钮文字
```

**视觉风格建议**：
- 🎨 配色方案：
  - 主色调：蓝色（#4299e1）
  - Hero背景：浅色渐变（#f5f7fa → #e8f0f8）
  - CTA背景：深色（#1a1a2e）
  - 文字颜色：#1a1a1a（深黑）、#666（中灰）、#999（浅灰）
  - 浅色背景：#f7fafc
- 📝 字体：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- ⚪ 圆角：8-12px小圆角（卡片）、12px（按钮）、20px（产品图片）
- 💫 阴影：产品图片阴影（0 20px 50px），卡片阴影（0 4px 12px）
- 🎯 对比：深色背景+白色文字形成强对比
- 🌟 动画：产品图片缩放（1.02）、卡片上移（-4px）

**适用场景**：
- ✅ 电商产品（实物商品）
- ✅ 硬件设备（产品展示）
- ✅ 订阅服务（价格展示）
- ❌ 不适合需要功能详解的产品（推荐template-01）
- ❌ 不适合品牌故事类产品（推荐template-03）

## 美学重点
- 🖼️ **产品视觉优先**：大图展示产品外观，视觉冲击力强
- 💰 **价格突出展示**：清晰的价格信息，引导转化决策
- 🎯 **强CTA转化导向**：深色背景+大号按钮，明确的购买引导
- 📐 **分屏布局平衡**：左文右图，信息与视觉的完美平衡
- 🌟 **蓝色专业主题**：统一的蓝色视觉语言，建立专业信任感
- 💫 **微交互细节**：产品图片缩放、卡片上移，增强页面活力
