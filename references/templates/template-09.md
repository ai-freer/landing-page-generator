# template-09: 趣味连接型

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**


**设计理念**：
引入全新的"视觉路径"(Visual Path)核心概念，强调动态的流转感。使用一条贯穿页面的虚线将各个板块串联起来，这是一个非常聪明的心理暗示设计，让用户感觉不是在看独立的"块"，而是在走一段"旅程"(Journey)。图片不再是裸露的，而是加上了明显的有色边框，增加了装饰感和界限感。

**结构组成**：
1. Hero区域（左右分屏，带框设计）
   - 左侧：标题、紫色主按钮、白色描边次按钮
   - 右侧：核心插画，外围包裹粗线条的紫色边框（4px solid #8B5CF6）
   - 边框效果：让图片看起来像是一个"窗口"或"屏幕"，突出产品的展示属性

2. Connected Flow区域（带连接线的交错布局）
   - 3个部分（What - Why - How），左右交错布局
   - 关键差异：图片与图片之间通过手绘风格的虚线连接
   - 虚线路径：从第一张图的底部延伸出来，弯曲绕过文字，连接到下一张图的顶部
   - SVG实现：绝对定位的SVG层（z-index: -1），动态绘制path
   - Framed Illustration：每张插画都被圆角矩形边框包裹（3px solid #8B5CF6），显得规整
   - Step Indicators：每部分都有圆形步骤指示器（数字1/2/3），暗示"步骤"或"逻辑递进"
   - 引导作用：引导用户一路向下滑动，降低跳出率

3. FAQ区域（卡片式问答）
   - 垂直卡片列表，不再是折叠面板
   - 每个问题被包裹在独立的白底圆角矩形框中，带有轻微的边框描边（2px solid #E5E7EB）
   - 标题居中："关于 [产品名] 的常见问题"
   - 悬停效果：边框变紫、阴影增强

4. Fat Footer（重型页脚）
   - 纯色块页脚：使用品牌主色（深紫色#7C3AED）作为底色
   - 左侧：Logo + Slogan + 版权信息
   - 右侧：多列链接组（产品、帮助、服务）
   - 视觉效果：给页面一个非常有力的收尾（Strong Finish），明确告诉用户"页面到底了，这里是所有资源的入口"

**核心特点**：
- 🎯 **引导性最强**：利用虚线把用户视线像过山车一样从头带到尾
- 🛤️ **视觉路径**：贯穿页面的虚线串联各个板块
- 🎨 **边框艺术**：所有图片和卡片都有有色边框（#8B5CF6）
- 🔢 **步骤指示**：圆形数字标识暗示流程递进
- 🎪 **趣味连接**：紫色主调 + 虚线路径，营造趣味感
- 📦 **重型页脚**：纯色块页脚，有力收尾

**适用场景**：
- 流程化工具（三步生成视频、四步完成注册等）
- 教程类产品（强调简单易用）
- 新手引导（步骤化展示）
- 需要降低跳出率的产品
- 强调流程的产品

**配置要点**：

**必填字段**：
```yaml
product:
  name: 产品名称
  tagline: 一句话价值主张
```

**Hero配置（可选）**：
```yaml
hero:
  image_url: 首屏右侧插图URL（可选）
```

**Connected Sections配置（新增）**：
```yaml
connected_sections:
  - title: 第一步标题
    description: 详细描述...
    image_url: 插图URL（可选）
  - title: 第二步标题
    description: 详细描述...
    image_url: 插图URL（可选）
  - title: 第三步标题
    description: 详细描述...
    image_url: 插图URL（可选）
```

**FAQ配置**：
```yaml
faq:
  - question: 常见问题1
    answer: 详细答案
  - question: 常见问题2
    answer: 详细答案
  # ... 建议4-6个问题
```

**可选字段**：
```yaml
footer:
  tagline: 品牌标语
```

**视觉风格建议**：
- 🎨 主色调：紫色（#8B5CF6）作为品牌主色
- 🖼️ 边框统一：`border: 2-4px solid #8B5CF6`，`border-radius: 12-24px`
- 🛤️ 虚线路径：SVG绝对定位，stroke-dasharray="8,8"，opacity=0.3
- 🔢 步骤指示：圆形数字（40px，紫色背景，白色数字）
- 📝 字体：干净的无衬线字体，标题加粗
- ⚪ 圆角：16-24px大圆角
- 💫 阴影：柔和的阴影效果
- 🎯 对比：白色卡片上的边框悬停效果
- 📐 响应式：移动端隐藏虚线路径（display: none）
- 🎪 趣味感：紫色边框+虚线路径营造趣味连接感

**技术实现要点**：
- ✅ SVG路径层：使用绝对定位的SVG，z-index: -1
- ✅ 响应式处理：移动端隐藏虚线路径（因为图片位置会变）
- ✅ 虚线样式：stroke-dasharray="8,8"，颜色#8B5CF6，透明度0.3
- ✅ 边框统一：所有图片和卡片使用相同的紫色边框
- ✅ 步骤指示器：圆形数字，与边框颜色呼应

**使用建议**：
- ✅ 适合流程化工具（步骤清晰）
- ✅ 适合教程类产品（强调简单易用）
- ✅ 适合需要降低跳出率的产品
- ✅ 适合新手引导（步骤化展示）
- ❌ 不适合复杂功能产品（推荐template-07/08）
- ❌ 不适合极简风格产品（推荐template-05）

## 美学重点
- 🛤️ **虚线路径串联**：SVG虚线连接各个步骤，引导用户自然浏览
- 🔢 **步骤指示器**：圆形数字标识，清晰展示流程进度
- 🎪 **趣味连接感**：紫色边框+虚线路径，营造轻松愉悦的浏览体验
- 💜 **统一紫色视觉**：边框、数字、路径使用统一紫色，强化品牌识别
- ⚪ **大圆角设计**：16-24px圆角，友好不尖锐
- 🌟 **降低认知负担**：步骤化展示，让复杂流程变得简单易懂
