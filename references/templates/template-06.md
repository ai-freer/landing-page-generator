# template-06: 极简插画风

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**


**设计理念**：
结合极简主义与手绘插画元素，通过Z型视觉流线和叙事型布局，将复杂信息转化为易于理解的视觉故事。特别适合创意工具、生活方式类产品或需要情感连接的品牌。

**结构组成**：
1. Hero区域（全屏白色背景，居中对齐）
   - 主标题：产品名称，使用柔和的渐变色
   - 副标题：简洁的价值主张
   - 主CTA按钮：圆角设计，柔和色彩
   - 手绘插画：抽象几何图形或产品示意图

2. Value Props区域（Zig-Zag布局，3个部分）
   - 每个部分采用左右交替的分屏布局
   - 一侧：文本说明（标题+描述）
   - 另一侧：对应的手绘插画
   - 视觉引导：从左到右、从上到下的Z型流动

3. Feature Grid区域（网格布局）
   - 标题："如何工作"
   - 3×2网格：6个功能点
   - 每个卡片：手绘图标+功能标题+简短描述

4. Testimonials区域（3个用户评价）
   - 用户头像（圆形）
   - 用户评价
   - 用户姓名和身份

5. FAQ区域（手绘风格）
   - 折叠式问答
   - 手绘装饰元素
   - 每个问题展开显示详细答案

6. Bottom CTA区域（居中对齐）
   - 标题：号召性语句
   - 副标题：补充说明
   - CTA按钮

7. Footer（极简设计）
   - 产品Logo/名称
   - 简短描述
   - 社交媒体链接

**核心特点**：
- ✨ 极简主义：留白充足，信息密度低
- 🎨 手绘插画：增加亲和力和创意感
- 📐 Z型布局：自然的视觉引导
- 🎭 情感化设计：通过故事和插画建立情感连接
- 🎯 叙事性：从问题到解决方案的完整故事线

**适用场景**：
- 创意工具（设计软件、写作工具等）
- 生活方式产品（健身App、冥想App等）
- 内容创作平台（播客、视频制作等）
- 社区类产品（兴趣小组、学习社区等）
- 品牌着陆页（需要情感共鸣的品牌）

**配置要点**：

**必填字段**：
```yaml
product:
  name: 产品名称
  tagline: 一句话价值主张
  description: 简短描述（2-3句）
```

**核心配置（zigzag_sections）**：
```yaml
zigzag_sections:
  - title: 第一部分标题
    description: 详细描述...
    image_url: 手绘插画URL
    image_position: right  # 或left，交替使用
  - title: 第二部分标题
    description: 详细描述...
    image_url: 手绘插画URL
    image_position: left
  - title: 第三部分标题
    description: 详细描述...
    image_url: 手绘插画URL
    image_position: right
```

**功能网格**：
```yaml
features:
  - title: 功能1
    description: 简短描述（1句话）
  - title: 功能2
    description: 简短描述
  # ... 共6个功能
```

**可选字段**：
```yaml
testimonials:
  - name: 用户姓名
    role: 身份/职业
    content: 评价内容
    avatar_url: 头像URL
  # ... 共3个评价

faq:
  - question: 常见问题1
    answer: 详细答案
  - question: 常见问题2
    answer: 详细答案

cta:
  title: 底部CTA标题
  subtitle: 补充说明
  button_text: 按钮文字

footer:
  tagline: 品牌标语
  social_links:
    - name: Twitter
      url: https://twitter.com/...
    - name: GitHub
      url: https://github.com/...
```

**视觉风格建议**：
- 🎨 配色：使用柔和的渐变色和低饱和度色彩
- ✏️ 插画风格：手绘线条、抽象几何、扁平化设计
- 📝 字体：干净的无衬线字体，大小对比明显
- 🖼️ 图片：手绘插画或高质量照片，避免过于写实的商业图
- ⚪ 圆角：按钮和卡片使用较大的圆角半径（16px+）
- 💫 阴影：柔和的阴影，避免硬边

**适用场景**：创意工具、生活方式App、内容创作平台、社区类产品

## 美学重点
- 🎨 **极简插画风**：手绘线条+抽象几何+扁平化设计，亲和力强
- 🖼️ **插画视觉核心**：插画作为主要视觉元素，传达轻松愉悦的品牌调性
- 📐 **Z型布局**：左图右文→右图左文交替，引导用户自然浏览
- ✏️ **手绘质感**：手绘线条、柔和阴影，营造温暖亲切感
- 🌟 **柔和配色**：低饱和度渐变色，避免刺眼，舒适自然
- 💫 **大圆角设计**：按钮和卡片大圆角（16px+），友好不尖锐
