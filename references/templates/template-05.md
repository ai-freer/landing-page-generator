# template-05: 深色赛博风

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**


**设计理念**：
深色背景+霓虹点缀的赛博朋克风格，适合科技产品和创意工具。采用动态渐变、发光效果和悬浮动画，营造未来科技感。

**结构组成**：
1. Hero区域（深色背景，居中对齐）
   - 产品名称（H1，72px，渐变文字效果）
   - 副标题（24px，灰色）
   - CTA按钮（渐变背景，发光效果，50px圆角）
   - Hero图片（可选，悬浮动画，发光边框）
   - 背景动画：旋转光晕
   - 作用：营造科技感氛围

2. Features区域（可选，网格布局）
   - 区域标题（48px，居中）
   - 副标题（18px，居中）
   - 功能卡片（3-6个）
     - 功能图标（深色背景，发光边框）
     - 功能标题（渐变文字）
     - 功能描述（灰色）
   - 悬停效果：卡片发光、渐变遮罩
   - 作用：展示核心功能

3. Testimonials区域（可选，卡片布局）
   - 区域标题（48px，居中）
   - 副标题（18px，居中）
   - 用户评价卡片
   - 作用：建立信任

4. CTA区域（深色背景）
   - CTA标题（渐变文字）
   - 副标题（灰色）
   - CTA按钮（渐变背景，发光效果）
   - 作用：引导转化

5. Footer区域（深色背景）
   - 品牌名称
   - 版权信息

**核心特点**：
- 🌟 深色主题：#0d1117背景，深色卡片（#1a1f2e）
- 🎨 霓虹配色：青色（#00f2ff）、紫色（#7c3aed）、蓝色（#3b82f6）
- ✨ 发光效果：按钮发光、卡片发光、边框发光
- 💫 动态动画：旋转光晕、悬浮动画、渐变遮罩
- 🎯 渐变文字：标题使用渐变文字效果

**配置要点**：

**必填字段**：
```yaml
product:
  name: 产品名称
  tagline: 一句话价值主张
```

**可选字段**：
```yaml
features:
  - title: 功能1
    description: 简短描述
  # ... 建议3-6个功能

mock_data:
  testimonials:
    - quote: 评价内容
      author: 作者信息
  # ... 建议3-6个评价

cta:
  button_text: 按钮文字
```

**视觉风格建议**：
- 🎨 配色方案：
  - 主色调：青色（#00f2ff）、紫色（#7c3aed）、蓝色（#3b82f6）
  - 渐变1：青色→紫色（#00f2ff → #7c3aed）
  - 渐变2：蓝色→青色（#3b82f6 → #00f2ff）
  - 背景色：深色（#0d1117）
  - 卡片背景：深灰（#1a1f2e）
  - 文字颜色：#ffffff（白色）、#9ca3af（灰色）
- 📝 字体：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- ⚪ 圆角：16px中圆角（卡片）、50px大圆角（按钮）、20px（Hero图片）
- 💫 阴影：深色阴影（0 30px 60px rgba(0,0,0,0.5)），发光效果
- 🎯 对比：深色背景+白色文字+霓虹渐变
- 🌟 动画：
  - 旋转光晕：20s线性循环
  - 悬浮动画：6s上下浮动
  - 按钮光泽：3s无限循环
  - 卡片发光：悬停时渐变遮罩
- 🎨 渐变：背景渐变、文字渐变、按钮渐变
- 🌐 发光：按钮发光（0 0 30px rgba(0,242,255,0.3)）、卡片发光边框

**适用场景**：
- ✅ 科技产品（AI、区块链、数据平台）
- ✅ 创意工具（设计软件、开发工具）
- ✅ 游戏产品（游戏平台、游戏工具）
- ❌ 不适合传统企业产品（推荐template-01）
- ❌ 不适合极简风格产品（推荐template-03）

## 美学重点
- 🌟 **深色赛博美学**：深色背景+霓虹渐变配色，强烈的科技感
- 💫 **动态光效**：旋转光晕、悬浮动画、按钮光泽，营造"活"的页面
- 🎨 **渐变文字和按钮**：青紫蓝三色渐变，视觉冲击力强
- ⚡ **发光效果**：按钮发光、卡片发光边框，增强赛博朋克氛围
- 🎯 **高对比度**：深色背景+白色文字，确保可读性的同时保持科技感
- 🌐 **动画丰富**：多种动画叠加，页面充满生命力和动态感
