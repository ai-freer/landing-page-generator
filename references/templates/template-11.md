# template-11: 便当盒式（Bento Grid）

## ⚠️ 核心原则
**本文档的"美学重点"章节是生成时的首要参考。HTML/CSS技术细节仅作为排版结构和命名规范参考，生成时应以美学高度和视觉冲击力为首要目标，不拘泥于代码细节。**


**设计理念**：
采用2025年流行的"Bento Grid"（便当盒式）布局，放弃传统的从上到下线性叙事，将功能模块像便当盒一样平铺在网格中，展示产品功能的丰富性和模块化。核心是"输入优先"的Hero区域，直接展示真实的交互界面，让用户一进来就看到闪烁的光标或正在自动键入的Prompt演示。

**设计风格**：Functional Chic（功能性时尚）
- Glassmorphism 2.0（磨砂玻璃2.0）：格子背景通常是深灰半透明，通过backdrop-filter: blur()透出背后的流动光晕
- Spotlight Effect（聚光灯效果）：鼠标滑过格子时，格子边缘或背景会有亮光跟随鼠标移动
- Ambient Glow（环境光晕）：背景有流动的渐变光晕动画，营造科技感

## 结构组成

### 1. Hero区域 - Input First（输入优先首屏）
- **结构定义**：输入优先的首屏设计
- **内容**：
  - 产品名称和标语
  - 巨大的、真实的交互输入框（InteractiveInputContainer）
  - 流式打字效果演示（StreamingTextResponse）
  - 输入建议快捷按钮
- **作用**：模拟产品真实界面，让用户一进来就直接看到产品如何工作

### 2. Bento Grid区域（全屏便当网格）
- **结构定义**：巨大的CSS Grid容器（4列布局）
- **网格布局**：
  - 大格（2x2）：核心功能演示视频或动态图表
  - 中格（1x2）：实时数据展示（如"已处理100w+任务"）
  - 小格（1x1）：关键特性Icon、用户头像、集成Logo
- **视觉特征**：
  - 每个格子都是独立的圆角矩形，带有微弱的边框和独立的背景光效
  - Spotlight Effect：鼠标悬停时，亮光跟随鼠标移动
- **作用**：展示产品功能的丰富性和模块化

### 3. Infinite Marquee（无限滚动条）
- **结构定义**：无限循环滚动条
- **内容**：合作伙伴/集成工具的Logo（Github、Notion、Linear等）
- **交互**：无限循环滚动
- **作用**：暗示AI的连接能力和生态系统

### 4. CTA区域（可选）
- **结构定义**：简洁的CTA提示框
- **视觉特征**：渐变背景和玻璃态效果
- **作用**：引导转化

## 技术特点
- 🎨 CSS Grid布局：`display: grid; grid-template-columns: repeat(4, 1fr); gap: 16px;`
- ✨ Spotlight Effect：使用radial-gradient追踪鼠标坐标更新CSS变量
- 🌊 Glassmorphism 2.0：`backdrop-filter: blur(20px)` + 半透明背景
- 💫 Ambient Glow：背景流动的渐变光晕动画
- ⚡ 流式打字效果：JavaScript实现的打字机动画
- 🎯 聚光灯效果：鼠标悬停时，亮光跟随鼠标移动
- 📊 动态图表：CSS动画实现的数据可视化

## 视觉风格建议
- 🎨 配色方案：
  - 主色调：紫色（#8b5cf6）+ 蓝色（#3b82f6）渐变
  - 强调色：青色（#22d3d1）+ 粉色（#ec4899）
  - 背景色：深色（#0a0a0a）+ 半透明玻璃态
  - 边框色：rgba(255, 255, 255, 0.1)
- 📝 字体：干净的无衬线字体，标题使用渐变文字效果
- 🖼️ 图片：高质量的产品截图或插画
- ⚪ 圆角：24px大圆角（卡片）+ 16px中圆角（按钮）
- 💫 阴影：柔和的阴影效果 + 聚光灯效果
- 📐 布局：4列网格布局（桌面端）→ 2列（平板端）→ 1列（移动端）
- 🎯 对比：深色背景+白色文字形成强对比

## 美学重点
- 📦 **模块化容器**：Bento Grid布局，独立圆角矩形，每个模块自成一体的视觉表达
- 🎯 **聚光灯效果**：鼠标悬停时亮光跟随移动，创造动态交互的视觉反馈
- 🌊 **玻璃态2.0**：半透明背景+backdrop-filter模糊，营造层次感和空间深度
- ⚡ **流式打字**：模拟真实AI输入输出，增强交互真实感
- 💫 **环境光晕**：背景流动渐变动画，营造呼吸感和生命力
- 🎨 **渐变文字**：标题使用渐变文字效果，提升视觉冲击力
- 🔄 **无限滚动**：Marquee的循环滚动，暗示无限可能和扩展性
